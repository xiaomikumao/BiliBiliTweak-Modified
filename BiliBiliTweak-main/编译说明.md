# BiliBiliTweak 编译说明（保留直播间人数和礼物特效）

## 修改说明

已修改 `BiliBiliTweak/src/Home/Live/NJLiveDetailAd.xm` 文件，注释掉了以下隐藏功能的代码：

1. **保留直播间右上角人数显示**
   - 注释掉了 `BBLiveBasePopularHotRankEntryView` 的 hook
   - 注释掉了 `BBLiveBasePopularRankEntryView` 的 hook
   - 注释掉了 `BBLiveVerticalPanelViewController` 的 `popularRankEntryViews` 方法

2. **保留礼物特效**
   - 注释掉了 `BBLiveBaseAnimationView` 的 hook（礼物动画视图）

## 编译环境要求

### 必需环境
- macOS 系统
- Xcode (建议 12.0+)
- [MonkeyDev](https://github.com/AloneMonkey/MonkeyDev) 或 [Theos](https://theos.dev/)

### 可选要求
- 越狱 iOS 设备（用于测试）
- iOS 开发证书（用于签名）

## 编译步骤

### 方法一：使用 Xcode + MonkeyDev（推荐）

1. **安装 MonkeyDev**
   ```bash
   sudo git clone https://github.com/AloneMonkey/MonkeyDev.git /opt/MonkeyDev
   sudo /opt/MonkeyDev/bin/md-install
   ```

2. **打开项目**
   - 双击 `BiliBiliTweak.xcodeproj` 用 Xcode 打开

3. **配置项目**
   - 选择目标设备（越狱设备或使用 Any iOS Device）
   - 设置签名证书（如果使用免费证书，需要手动签名）

4. **编译项目**
   - 点击 Xcode 的 Build 按钮（▶️）
   - 或菜单 Product → Build

5. **获取编译产物**
   - 编译成功后，在 Products 目录下找到 `BiliBiliTweak.dylib`
   - 右键点击 → Show in Finder

### 方法二：使用 Theos 命令行编译

1. **安装 Theos**
   按照 [Theos 官方文档](https://theos.dev/docs/installation) 安装

2. **编译**
   ```bash
   cd BiliBiliTweak
   make
   ```

3. **打包**
   ```bash
   make package
   ```

## 安装方法

### 越狱设备安装

1. **使用 Cydia**
   - 将编译好的 `.deb` 文件传输到设备
   - 使用 Filza 等文件管理器安装
   - 或使用 Sileo/ Cydia 直接安装

2. **使用 SSH**
   ```bash
   scp BiliBiliTweak.dylib root@设备IP:/Library/MobileSubstrate/DynamicLibraries/
   ssh root@设备IP
   killall -9 BiliBili
   ```

### 非越狱设备（巨魔注入）

1. **使用 TrollStore**
   - 将 `.dylib` 文件注入到 BiliBili APP
   - 使用 [Azule](https://github.com/Al4ise/Azule) 等工具打包

2. **使用 MonkeyDev 自动注入**
   - 在 Xcode 中直接运行到设备

## 注意事项

1. **代码签名**
   - iOS 要求所有可执行文件必须签名
   - 越狱设备可以使用 `ldid` 工具签名
   ```bash
   ldid -S BiliBiliTweak.dylib
   ```

2. **依赖框架**
   - 确保设备已安装 `CydiaSubstrate`
   - 可以通过 Cydia/ Sileo 安装 `mobilesubstrate`

3. **版本兼容**
   - 插件支持 BiliBili 8.41.0+
   - 不同版本可能需要不同的 offset

## 故障排除

### 编译失败
- 检查 MonkeyDev/ Theos 是否正确安装
- 检查 Xcode 命令行工具是否正确配置
- 查看编译日志，确认缺少的依赖

### 插件不生效
- 确认 dylib 已正确安装到 `/Library/MobileSubstrate/DynamicLibraries/`
- 重启 BiliBili APP
- 检查设备日志：`tail -f /var/log/syslog | grep BiliBiliTweak`

### 崩溃问题
- 可能是版本不兼容，尝试更新插件
- 检查是否与其他插件冲突
