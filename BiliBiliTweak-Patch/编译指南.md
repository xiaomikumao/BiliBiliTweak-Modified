# BiliBiliTweak 补丁编译和使用指南

## 简介
这个补丁用于恢复 BiliBiliTweak 隐藏的直播间人数和礼物特效功能。

## 文件说明
- `Tweak.xm` - 补丁源码，hook 相关类并恢复功能
- `control` - deb 包元信息
- `Makefile` - Theos 编译配置
- `.github/workflows/build.yml` - GitHub Actions 自动编译工作流

## 编译方法（无 Mac 环境）

### 方法一：使用 GitHub Actions 自动编译（推荐）

1. **创建 GitHub 仓库**
   - 登录 GitHub，创建一个新的仓库（比如叫 `BiliBiliTweakPatch`）
   - 选择 Public（公开）仓库，这样可以使用免费的 Actions

2. **上传源码**
   - 将 `BiliBiliTweak-Patch` 目录下的所有文件上传到 GitHub 仓库
   - 包括 `.github` 目录（确保 GitHub Actions 工作流被识别）

3. **触发编译**
   - 进入仓库的 "Actions" 标签页
   - 选择 "Build BiliBiliTweak Patch" 工作流
   - 点击 "Run workflow" 手动触发编译

4. **下载编译好的 deb**
   - 等待编译完成（大约 5-10 分钟）
   - 在 Actions 页面的右侧 "Artifacts" 区域，下载 `BiliBiliTweakPatch.deb`

### 方法二：使用 GitHub Codespaces 编译

1. **创建 GitHub Codespaces**
   - 在仓库页面点击 "Code" 按钮
   - 选择 "Codespaces" 标签
   - 点击 "Create codespace on main"

2. **在 Codespaces 中编译**
   ```bash
   # 安装 Theos
   git clone --recursive https://github.com/theos/theos.git ~/theos
   export THEOS=~/theos
   export PATH=$THEOS/bin:$PATH
   
   # 下载 iOS SDK
   mkdir -p $THEOS/sdks
   curl -Lo $THEOS/sdks/iPhoneOS16.5.sdk.tar.gz https://github.com/theos/sdks/raw/master/iPhoneOS16.5.sdk.tar.gz
   tar -xzf $THEOS/sdks/iPhoneOS16.5.sdk.tar.gz -C $THEOS/sdks/
   
   # 编译
   make package FINALPACKAGE=1
   ```

3. **下载 deb 文件**
   - 编译完成后，deb 文件在 `packages/` 目录下
   - 通过 Codespaces 的文件管理器下载

## 安装方法

1. **将 deb 文件传输到越狱设备**
   - 通过 SSH、iFile、Filza 等工具

2. **安装 deb**
   ```bash
   dpkg -i BiliBiliTweakPatch.deb
   ```

3. **重启 BiliBili APP**
   - 彻底关闭 BiliBili APP（划掉后台）
   - 重新打开

## 工作原理

补丁插件通过以下方式恢复功能：
1. Hook `BBLiveBasePopularHotRankEntryView` - 恢复直播间右上角热门排行
2. Hook `BBLiveBasePopularRankEntryView` - 恢复直播间右上角排行
3. Hook `BBLiveBaseAnimationView` - 恢复礼物特效
4. 强制设置 `hidden = NO`，阻止原插件隐藏这些视图

## 注意事项

1. **加载顺序**：补丁插件需要在原插件之后加载
   - 通过延时 1 秒加载确保后执行
   - 如果需要，可以重命名 dylib 文件确保按字母顺序后排

2. **兼容性**：
   - 仅测试于 BiliBiliTweak 3.0.0
   - 如果 BiliBili APP 更新，可能需要更新补丁

3. **卸载**：
   ```bash
   dpkg -r com.touchWorld.bilibiliTweak.patch
   ```

## 备选方案：直接修改原插件（需要 Mac）

如果你有 Mac 环境，可以直接修改原插件源码：

1. 修改 `BiliBiliTweak/src/Home/Live/NJLiveDetailAd.xm`
2. 注释掉隐藏直播间功能的代码
3. 使用 Xcode + MonkeyDev 编译

## 常见问题

**Q: 补丁安装后没有效果？**
A: 确保原插件已安装，且补丁后安装。尝试重启设备。

**Q: 编译失败？**
A: 检查 GitHub Actions 日志，确保 iOS SDK 下载成功。

**Q: 能否直接修改原 deb 文件？**
A: 原 deb 中的 dylib 是编译好的二进制文件，无法直接修改。需要重新编译。
