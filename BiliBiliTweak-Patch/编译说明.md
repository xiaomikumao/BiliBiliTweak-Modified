# BiliBiliTweak Patch 编译说明

## 功能说明

这是一个 **BiliBiliTweak 的补丁插件**，用于恢复被原版插件隐藏的功能：

✅ **恢复直播间右上角人数显示**
- 人气榜入口
- 人气排行榜
- 用户排行榜

✅ **恢复礼物特效**
- MP4 动画
- Spine 动画
- SVGA 动画
- 礼物面板

## 工作原理

这个补丁会在 **BiliBiliTweak 之后加载**，通过以下方式恢复功能：

1. **不返回 nil** - 让原本的初始化方法正常执行
2. **不移除视图** - 保留被隐藏的 UI 元素
3. **调用原始实现** - 使用 `%orig` 调用原始代码

## 编译环境

### 必需
- macOS 系统
- Xcode (建议 12.0+)
- Theos 或 MonkeyDev

### 可选
- 越狱 iOS 设备（测试用）

## 编译步骤

### 方法 1：使用 Theos 命令行编译

```bash
# 设置 Theos 环境
export THEOS=/opt/theos
export PATH=$THEOS/bin:$PATH

# 进入项目目录
cd BiliBiliTweak-Patch

# 编译
make clean
make package FINALPACKAGE=1

# 编译成功后，.deb 文件会在 packages/ 目录下
```

### 方法 2：使用 Xcode + MonkeyDev

1. 创建新的 MonkeyDev 项目
2. 复制 `Tweak.xm` 和 `Makefile` 的内容
3. 编译并运行

### 方法 3：使用 GitHub Codespaces（自动）

我已经创建了自动编译脚本，运行：

```bash
bash build_patch.sh
```

## 安装方法

### 1. 安装依赖

**必须先安装原版 BiliBiliTweak！**

- 添加源：https://touchfriend.github.io
- 安装 BiliBiliTweak

### 2. 安装补丁

**越狱设备：**
```bash
# 使用 dpkg 安装
dpkg -i com.yourname.bilibilitweakpatch_1.0.0_iphoneos-arm.deb

# 重启 BiliBili
killall -9 BiliBili
```

**使用 Filza：**
1. 将 `.deb` 文件传输到设备
2. 使用 Filza 打开并安装
3. 重启 BiliBili APP

### 3. 验证安装

安装后，在 BiliBili 中：
1. 进入任意直播间
2. 查看右上角是否显示人数
3. 查看是否有礼物特效

## 故障排除

### 补丁不生效

1. **检查加载顺序**
   ```bash
   # 在设备上查看日志
   tail -f /var/log/syslog | grep BiliBiliTweakPatch
   ```
   
   应该看到：
   ```
   [BiliBiliTweakPatch] 补丁已加载！
   ```

2. **确保原版已安装**
   - 补丁依赖原版 BiliBiliTweak
   - 如果原版未安装，补丁不会生效

3. **尝试调整加载顺序**
   - 补丁的 plist 文件应该包含原版 bundle ID
   - 确保在原版之后加载

### 崩溃问题

如果安装后 BiliBili 崩溃：

1. **类名可能不正确**
   - 不同版本的 BiliBili 可能使用不同的类名
   - 查看崩溃日志，确认类名

2. **暂时禁用补丁**
   ```bash
   # 重命名 dylib 文件
   mv /Library/MobileSubstrate/DynamicLibraries/BiliBiliTweakPatch.dylib \
      /Library/MobileSubstrate/DynamicLibraries/BiliBiliTweakPatch.dylib.disabled
   
   # 重启 BiliBili
   killall -9 BiliBili
   ```

## 自定义修改

### 添加更多恢复功能

编辑 `Tweak.xm`，添加更多 hook：

```objective-c
%hook ClassName

- (id)initWithFrame:(CGRect)frame {
    return %orig;  // 恢复原始行为
}

%end
```

### 只恢复特定功能

注释掉不需要的 hook，只保留想要的功能。
